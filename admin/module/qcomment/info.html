Documentation of qComment

Introduction

This module is allow users to add comments or rates to an article.
You can use qComment for anything you want to comment/rate, any modules, any pages, anything.
This module is unique, this module is not a stand alone, instead called from another module (for example qStory & Micro Gallery use this module).
NOTE: Some modules may require this module to accept comments & ratings.

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

After installing, you have to configure it before you can start using it. See Configuring section for more information.

You may also need to change your template, see Usage section.

Please merge /admin/module/qcomment/style.css to your existing stylesheet file.

You may need to modify some language vars. Open /module/qcomment/window.php & main.php to see the vars, and make approriate changes.

Configuring

Before you can use this Module, you have to configure it first.

  1. Go to Admin CP > Configure Site > Modules.
  2. Find qComment, and click Configure.

Usage

To display user rating (and form), simply use this:

  • Code:
    <!-- BEGINMODULE qcomment -->
    mode = rate
    mod_id = $module_id
    item_id = $item_id
    <!-- ENDMODULE -->
  • Parameters:
    $module_id = your module id
    $item_id = a unique id (story id, item id, image id, etc)
  • For example, to display rate box for module qStory (module id: qstory), story id 5, use:
    Code:
    <!-- BEGINMODULE qcomment -->
    mode = rate
    mod_id = qstory
    item_id = 5
    <!-- ENDMODULE -->

To display user comment (and a link to form), simply use this:

  • Code:
    <!-- BEGINMODULE qcomment -->
    mode = comment
    mod_id = $module_id
    item_id = $item_id
    sort = $sort_mode
    title = $title
    items = $n
    <!-- ENDMODULE -->
  • Parameters:
    $module_id = your module id
    $item_id = a unique id (eg. story id, item id, image id, etc)
    $sort_mode = can be 'latest' or 'first' (default: latest)
    $title = item title (eg. my story, his story, etc, usually article title)
    $items = number of comment to display in window mode (default: 5)
  • For example, to display last 8 comments for module qStory (module id: qstory), story id 5, use:
    Code:
    <!-- BEGINMODULE qcomment -->
    mode = comment
    mod_id = qstory
    item_id = 5
    sort = latest
    title = Some Title
    items = 8
    <!-- ENDMODULE -->

Modification

You can modify this module output, by editing:

  • skins/default/module_qcomment_window.tpl
  • and, skins/default/module_qcomment.tpl
If you have multiple skins, you can copy those files to your skin's folder. Otherwise, it will use default skin.

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find qComment. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
