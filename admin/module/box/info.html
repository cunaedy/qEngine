Documentation of Box

Introduction

This module provide an easy solution to display static html of anything you want to the page (in the module locations) without editing the .tpl files. Best used with qE 4.0's module manager.

For example you can display Google AdSense, Bing Maps, or any other widgets to your site.

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

You may also need to change your template, see Usage section.

Usage

This module is intended to be used with Module Manager.

  • Open Module Manager
  • Select any module location you wish to display the module at.
  • Select this module. Click Save.
  • Edit this module configuration, and copy-paste the code you want, directly to the module configuration window.

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find Page Gallery. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
