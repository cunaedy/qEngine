Documentation of Multi Page

Introduction

This module allow you to combine multiple pages (by using page gallery) and other modules (eg Slide Show) into one single page.

While you may be able to achieve the same effect by using default Page Editor, but Multi Page have more features:

  • Cached whole page to improve page performance, instead of per element.
  • Page can be displayed without side bar.

To install this this module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Modules.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

Usage

To add a new multi page, from ACP, click Modules > Multi Page > Manage Multipage.

Edit the content, eg:

<div class="container">
	<div class="row">
		<div class="col-sm-6">
			<!-- BEGINMODULE page_gallery -->
			page_id = 2
			body = 1
			<!-- ENDMODULE -->
		</div>
		<div class="col-sm-6">
			<!-- BEGINMODULE page_gallery -->
			page_id = 3
			body = 1
			<!-- ENDMODULE -->
		</div>
	</div>
	<div class="row">
		<div class="col-sm-6">
			<!-- BEGINMODULE page_gallery -->
			page_id = 4
			body = 1
			<!-- ENDMODULE -->
		</div>
		<div class="col-sm-6">
			<!-- BEGINMODULE slideshow -->
			theme=theme-welcome
			cat_id=82
			<!-- ENDMODULE -->
		</div>
	</div>
</div>

Uninstalling

To uninstall this module, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find this module. Click Uninstall.
