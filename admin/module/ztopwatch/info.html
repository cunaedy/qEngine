Documentation of Ztopwatch

Introduction

This module replace the old stopwatch in qEngine v1.x, you know the "This page is generated in 0.10 second | 7 queries".
Because of the qEngine's processing procedure, using old stopwatch doesn't calculate the time needed to build modules.
So we build a new stopwatch (ztopwatch) to also calculate time needed to build modules. This should give more accurate information.
You should put this module as the last module on the page, ie. the bottom of the page.

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

You may also need to change your template, see Usage section.

Usage

  • Open a skin file, preferred outline.tpl, eg. skins/default/outline.tpl
  • At the bottom of the page, add:
  • Code:
    <!-- BEGINMODULE ztopwatch --><!-- ENDMODULE -->

Modification

To modify the text, formatting, etc, see /module/ztopwatch/window.php

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find Ztopwatch. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
