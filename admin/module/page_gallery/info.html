Documentation of Page Gallery

Introduction

Use this module to display your page (see Admin CP > Contents > Pages) in your site. For example, you may want to display a message in a specific area of your site, see welcome.tpl for a better example. You can display as many pages as you need.

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

You may also need to change your template, see Usage section.

Usage

  1. Open a skin file
  2. Any where, add:
  3. <!-- BEGINMODULE page_gallery -->
    option = value
    <!-- ENDMODULE -->
  4. Where option can be:
    • group_id = id of page type you want to display, eg. 'news', or 'genpg'. See Page Group Editor for more ID.
    • cat_id = id for a page category you want to display, eg. 1 for general category. See Page Category Editor for more ID.
    • page_id = id for a page you want to display, eg. 1 for welcome page. See Page Editor for more ID.
    • title = 1 | 0, use 1 ti display title, 0 (or ignore it) to hide it. Optional.
    • summary = 1 | 0, use 1 ti display summary, 0 (or ignore it) to hide it. Optional.
    • body = 1 | 0, use 1 to display body content, 0 (or ignore it) to hide it. Optional.
    • link = 1 | 0, use 1 to display link, 0 (or ignore it) to hide it. Optional.
    • image = 1 | 0, use 1 to display page main image, 0 (or ignore it) to hide it. Optional.
    • thumb = 1 | 0, use 1 to display page thumbnail image, 0 (or ignore it) to hide it. Optional.
    • date = 1 | 0, use 1 to display page creation date, 0 (or ignore it) to hide it. Optional.
    • time = 1 | 0, use 1 to display page creation time, 0 (or ignore it) to hide it. Optional.
    • limit = limited displayed items, default is 10. Optional.
    • offset = offset displayed items, default is 0. Optional.
    • orderby = display page list ordered by this value: page_title, page_date, page_unix, page_rating, page_comment and/or page_hit. Optional.
    • sort = display page list ordered by this method: ASC for ascending, and DESC for descending. Optional.
    • style = default | list, default is default. Optional.
    • sectiontpl = name of section to use as output template. Optional.
    • NOTES: Almost all parameters are optional, the only required parameter is either: group_id, cat_id, or page_id.
    • NOTES 1: group_id, cat_id & page_id is mutually exclusive. Use only one, or none at all to display all contents.
    • NOTES 2: you can also assign multiple values for group_id, cat_id & page_id by using comma, eg: 1,2,3.
    • NOTES 3: you can also assign multiple values for orderby, eg: page_title, page_date.

Examples:

To display a welcome page:

<!-- BEGINMODULE page_gallery -->
page_id = 1
body = 1
<!-- ENDMODULE -->

To display list of pages from NEWS page type:

<!-- BEGINMODULE page_gallery -->
group_id = news
title = 1
summary = 1
style = list
<!-- ENDMODULE -->

To display list of latest entries from your BLOG page type (assuming your BLOG category id are: 5, 6, 7):

<!-- BEGINMODULE page_gallery -->
cat_id = 5, 6, 7
title = 1
summary = 1
date = 1
time = 1
link = 1
orderby = page_unix
sort = desc
<!-- ENDMODULE -->

To use custom template to display output:

<!-- BEGINMODULE page_gallery -->
group_id = news
tplsection = mytpl
<!-- ENDMODULE -->

<!-- BEGINSECTION mytpl -->
Define your template here
You can use any tags without enabling them in module
{$page_title}
{$page_summary}
{$more}
For more tags, please refer to the qe_page table structure.
<!-- ENDSECTION -->

Modification

To modify the text, formatting, etc, see /module/page_gallery/window.php

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find Page Gallery. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
