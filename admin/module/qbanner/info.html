Documentation of qBanner

Introduction

Use this module to upload & display simple banners. By 'simple' means no 3rd party administration, no time limit, no click tracking, no stats, nothing! Just upload & display!

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

You may also need to change your template, see Usage section.

Usage

  • Open a skin file, preferred outline.tpl, eg. skins/default/outline.tpl
  • At the bottom of the page, add:
  • Code:
    <!-- BEGINMODULE qbanner --><!-- ENDMODULE -->
    This will display any banner (random) from any banner type.
  • You can also specify which banner type to display, use:
    <!-- BEGINMODULE qbanner -->
    banner_type=X
    <!-- ENDMODULE -->
    Where X is banner type (A to E)
  • The fun part is, even if you specify a same banner_type for two locations, you won't see same ad on both places.

Modification

To modify the formatting, see /skins/default/module_qbanner.tpl

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find qBanner. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
